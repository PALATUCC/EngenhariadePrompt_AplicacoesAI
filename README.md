# Engenharia de Prompt - Aplicações em IA

## 📋 Descrição

Repositório dedicado às atividades práticas da disciplina de **Engenharia de Prompt e Aplicações em IA**. Este projeto contém materiais, exercícios e soluções para o aprendizado prático sobre técnicas de engenharia de prompts aplicadas ao desenvolvimento de aplicações com Inteligência Artificial.

## 🎯 Objetivo

Desenvolver competências em:
- ✅ Design e otimização de prompts para modelos de IA
- ✅ Aplicações práticas de engenharia de prompts
- ✅ Desenvolvimento de soluções low-code e no-code
- ✅ Integração de IA em aplicações reais

EngenhariadePrompt_AplicacoesAI/

## 🚀 Conteúdo por Unidade

### Unidade 3: Aplicações Práticas

#### **Low-Code** 📱
Atividades que combinam programação minimal com ferramentas de IA para criar soluções eficientes.

#### **No-Code** 🤖
Atividades utilizando plataformas e ferramentas sem necessidade de programação tradicional.

## 💡 Tecnologias e Ferramentas

Este repositório explora o uso de:
- Modelos de Linguagem (LLMs)
- Prompts Engineering
- Ferramentas Low-Code
- Plataformas No-Code
- APIs de IA

## 📝 Como Usar Este Repositório

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/PALATUCC/EngenhariadePrompt_AplicacoesAI.git
   cd EngenhariadePrompt_AplicacoesAI
   ```

2. **Navegue pelas unidades:**
   - Acesse a pasta `Unidade3` para atividades práticas
   - Escolha entre abordagens Low-Code ou No-Code

3. **Complete as atividades:**
   - Leia as instruções em cada arquivo `.md`
   - Implemente os exercícios propostos
   - Teste suas soluções
